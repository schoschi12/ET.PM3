Cable-Monitor Code Documentation

This documents the code for the Electronics-Project Modules ET.PM3: Cable-Monitor
in the third year of the part-time electronics engineering curriculum at the School of Engineering of the Zurich University of Applied Sciences.


measuring.c initiates the ADCs and performs measurements as well as implementing some filter functions.
menu.c initiates the available menus
pushbutton.c handles external interrupts.
distance.c handles the measurements of the E-field and displays the results.
strommessung.c handles the measurements of the B-field and displays the results.
main.c puts everything together.

Tests have been performed to ensure the functionality of all implemented features. They are documented here: testing.html

The code is written for the evaluation board

32F429IDISCOVERY (Discovery kit with STM32F429ZI MCU).
